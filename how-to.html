üìù Step-by-Step Instructions

Prefer written instructions? Here's what to do:

  1. Visit Anthropic's Console
    Go to console.anthropic.com and sign up for an account or log in if you already have one.
  2. Navigate to API Keys
    Once logged in, find the "API Keys" section in your dashboard. This is usually in the left sidebar or under your account settings.
  3. Create New API Key
    Click "Create Key" or "Generate API Key". Give it a descriptive name like "RevBot Plugin" so you can identify it later.
  4. Copy Your Key
    Important! Copy your API key immediately and store it safely. You won't be able to see it again after you close the window.
  5. Add to RevBot
    Open RevBot in Adobe After Effects, navigate to Settings, and paste your API key in the designated field. Save, and you're ready to go!
